文件结构

  • ./include: 头文件
    • Queue.h: 队列的抽象类
    • seqQueue.hh: 循环队列
    • vecQueue.hh: 顺序队列
    • linkQueue.hh: 链接队列
  • ./src: 测试例程
    • test0.cc：测试0 - 测试上述三种队列的基本功能，包括构造、入队和出队。
  • ./doc: HTML和LATEX文档
    • HTML文档打开方式：
cd ./path/to/projectRoot
./doc/html/index.html

Compiling

将./include文件夹加入IncludePath，编译./src文件夹中的单个源文件。

  • 例 g++编译并运行test0.cc
cd ./path/to/projectRoot
mkdir bin
g++ ./src/test0.cc -o ./bin/test0.exe -g -fexec-charset=GBK -std=c++11 -I ./include
./bin/test0.exe
